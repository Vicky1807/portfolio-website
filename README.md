# 🚀 Vignesh | Portfolio Website

This repository contains the source code for my personal portfolio website, hosted using **Azure Static Web Apps** and deployed via **GitHub Actions**.

The site showcases my background as an **ETRM Business & Data Analyst** and **Azure Certified Data Scientist**, along with selected projects and skills.

🔗 https://lively-ground-03f45c81e.2.azurestaticapps.net/

---

## 🧠 About the Project

This is a **static portfolio website** built using:

- HTML
- CSS

The project follows a **GitHub-first deployment approach**, where:

- GitHub is the source of truth and code
- Azure automatically redeploys the site on every push to the `main` branch

---

## 🏗 Architecture Overview

```
Local Machine (VS Code)
        ↓
     Git Commit
        ↓
     GitHub Repository
        ↓
 GitHub Actions (CI/CD)
        ↓
Azure Static Web Apps
```

---

## ☁️ Hosting & Deployment

- **Hosting**: Azure Static Web Apps (Free Tier)
- **CI/CD**: GitHub Actions (auto-generated by Azure)
- **SSL**: Enabled by default
- **Deployment Trigger**: Push to `main` branch
